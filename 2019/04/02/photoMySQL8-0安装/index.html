MySQL-8.0.15安装教程

Created2019-04-02Updated2019-05-17

轻松安装MySQL，避免误区

下载MySQL

点击此处进入MySQL下载地址

安装

1.解压
2.配置环境变量

计算机->属性->高级系统设置->环境变量->系统变量 ->在变量名为：Path 下添加：D:\mysql\mysql-8.0.15-winx64\bin
mysql目录+\bin
注意：最好在开始处或结尾添加 有的环境下可能需要添加;

3.打开cmd

以管理员身份运行cmd，并切换到mysqld的bin目录下，
cd /D D:\mysql\mysql-8.0.15-winx64\bin

4.初始化

键入mysqld --initialize --console
这时候需要耐心等待一会儿.
直到命令行出现D:\mysql\mysql-8.0.15-winx64\bin>
在命令行中找到
A temporary password is generated for root@localhost: (你的密码)

5.安装

键入 mysqld -install
提示sucess即为mysql服务安装成功

6.启动服务

键入 net start mysql

7.登入mysql账户

键入mysql -u root -p //root为你的用户名，-p后也可以直接输入密码

8.更改密码

键入alter user 'root'@'localhost' identified with mysql_native_password by '密码';

9.测试

键入show databases;

到这里这里呢，MySQL的安装教程就算完成了
